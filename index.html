Preguntas sobre CSS

  1. ¿Qué es CSS?

    CSS (Cascading Style Sheets) es un lenguaje de estilo utilizado para describir la presentación de un documento HTML. Permite separar el contenido de la presentación y facilita el diseño visual de las páginas web.

  2. ¿Cuál es la función principal de CSS?

    La función principal de CSS es controlar el aspecto visual del contenido HTML, como los colores, fuentes, márgenes, alineaciones, etc.

  3. ¿Cómo se puede incluir CSS en un documento HTML?

    CSS se puede incluir de tres maneras: en línea, interno y externo. La mejor práctica es utilizar archivos CSS externos para mantener la estructura del código organizada.

  4. ¿Qué son los selectores en CSS?

    Los selectores en CSS se utilizan para seleccionar los elementos HTML a los que se aplicarán los estilos. Pueden ser etiquetas, clases, identificadores, entre otros.

  5. ¿Qué son las propiedades y valores en CSS?

    Las propiedades son las características que se desean cambiar (color, fuente, márgenes) y los valores son los ajustes asignados a esas propiedades (rojo, Arial, 10px, etc.).

  6. ¿Cómo se aplica el modelo de caja en CSS?

    El modelo de caja en CSS es una forma de visualizar el diseño de un elemento HTML. Cada elemento se representa como una caja que contiene márgenes, bordes, rellenos y el contenido.

  7. ¿Qué es el "cascading" en CSS?

    El "cascading" en CSS define cómo se aplican los estilos cuando existen conflictos. CSS sigue un orden jerárquico y utiliza la cascada para determinar qué reglas tienen prioridad.

  8. ¿Qué son las media queries?

    Las media queries son reglas en CSS que permiten aplicar estilos dependiendo del tamaño de la pantalla o dispositivo. Son fundamentales para el diseño responsivo.

  9. ¿Cuáles son algunos estilos comunes que se pueden aplicar?

    Algunos estilos comunes incluyen la modificación de colores, fuentes, márgenes, rellenos, bordes, tamaños de imágenes, y alineaciones de texto.

  10. ¿Cómo se puede usar CSS para crear un diseño responsivo?

    Se puede usar CSS para crear un diseño responsivo aplicando media queries, que permiten ajustar los estilos para diferentes tamaños de pantalla (por ejemplo, dispositivos móviles y tablets).
